# Site web de recettes de cuisines : FindRecipe

Il s'agit de mon projet de fin d'année de L3 Info à l'UPHF.

Sa particularité est qu'il contient une base de données dans l'API node.js que j'ai conçu pour le site.

Un système d'abonnement et de création/connexion de comptes sont à prévoir.


Pour l'ouvrir, téléchargez l'intégralité des fichier depuis Code > Download Zip

Puis ouvrez index.html après avoir lancer la commande node server.js depuis le répertoire findARecipe pour lancer le serveur.
