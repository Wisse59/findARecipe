# Site web de recettes de cuisines : FindRecipe

Il s'agit de mon projet de fin d'année de L3 Info à l'UPHF.

Sa particularité est qu'il contient une base de données dans l'API node.js que j'ai conçu pour le site.

Il est possible de créer un compte et de se connecter afin de s'abonner aux catégories qui nous intéressent.
Un vrai e-mail nous est envoyé lorsqu'un plat qui en fait partie est ajouté au site, cela grâce à l'outil nodemailer.

Pour des questions de sécurité, notre mot de passe doit contenir au moins 10 caractères, 1 maj., 1min., 1 nombre et un car. spécial.
L'e-mail quant à lui doit prendre la forme : texte + @ + texte + . + texte pour être accepté.

Pour lancer le projet, veuillez télécharger l'intégralité des fichier depuis Code > Download Zip.
Ouvrez ensuite index.html sur votre navigateur.
Et enfin, lancez commande "node server.js" depuis le répertoire findARecipe, afin de lancer le serveur.
